# Quickstart: UI & Player

## Prerequisites
- Backend API (`/api/play`, Supabase) must be running/available.

## Running the Frontend

1. **Install Dependencies** (including new ones):
   ```bash
   npm install react-virtuoso zustand
   ```

2. **Start Dev Server**:
   ```bash
   npm run dev
   ```

3. **Verify Layout**:
   - Open `http://localhost:3000`.
   - Check Sidebar (Desktop) or Hamburger Menu (Mobile).
   - Check Player Bar fixed at bottom.

4. **Test Playback**:
   - Go to Charts.
   - Click a song.
   - Verify audio plays and player bar updates.
